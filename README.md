# BigContest
---
Our neighborhood franchisees, catch the signs of crisis early!
Data From: https://www.bigcontest.or.kr/

## 📖 Introduce
---
Analyzing franchise data during 2023~2024, predict which franchise store will be at risk after July 2025.
